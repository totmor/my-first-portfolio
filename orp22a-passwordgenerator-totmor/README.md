# Password generator

## Situatieschets

In deze oefening is het de bedoeling om een **paswordgenerator**  applicatie te ontwikkelen met behulp van Object Oriented.

## Gestelde eisen

Om de karakters uit het paswoord te bepalen herhaal je telkens volgende stappen:

* Bepaal een willekeurig getal overeenkomstig met een welbepaalde letter (cfr. ASCIItabel).

* Zet dit willekeurig getal om in de overeenkomstige letter.

* Bepaal willekeurig of het een hoofdletter of kleine letter moet worden. Gebruik hiervoor de juiste String-functie.

* Voeg de letter aan het paswoord toe.

* Bepaal de lengte van het paswoord door een willekeurig getal te genereren binnen het interval bepaald door de twee textboxen.

* Zorg voor een correcte foutafhandeling.

## Opdracht

1. Alle code welke nodig is voor het genereren van een paswoord moet zich in de klasse bevinden.

2. De UI mag alleen code bevatten om het businessmodel (klasse) aan te spreken.

## Uitbreiding

1. De waarde voor de minimum en maximum lengte worden ingesteld aan de hand van 2 sliders.
